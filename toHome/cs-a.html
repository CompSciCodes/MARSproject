Everyone needs a password. Program an algorithm in which users can answer a series of questions and get generated passwords for various websites and programs. All inhabitants will use the same algorithm but will determine their own unique passwords depending on how they answer the questions. No one is to share this information with anyone else. See this example of how your algorithm should work, but you must create your own unique session with the users. The password must be at least 8 characters long with symbols, capital and lowercase letters.

This will be crucial for the survival of anyone driving those routes!

Create your Java program using one of the three online compilers provided in the instructions, and be sure to include all the proper calculations so all the results are accurate.
